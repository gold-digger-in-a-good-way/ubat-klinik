# Sistem Pengambilan Ubat Klinik (PWA)

Projek ini merupakan sistem pengambilan ubat untuk klinik kerajaan yang dibangunkan menggunakan HTML, CSS dan JavaScript serta diintegrasikan dengan konsep Progressive Web App (PWA).

## Objektif Sistem
- Memudahkan pesakit menerima peringatan pengambilan ubat
- Mengurangkan kesesakan di kaunter farmasi
- Menyokong digitalisasi perkhidmatan klinik kerajaan

## Teknologi Digunakan
- HTML5
- CSS3
- JavaScript
- Progressive Web App (PWA)
- GitHub Pages

## Cara Menggunakan Sistem
1. Buka pautan sistem melalui browser
2. Masukkan nama ubat dan masa pengambilan
3. Tekan butang "Set Reminder Ubat"
4. Benarkan notifikasi
5. Sistem akan menghantar peringatan mengikut masa yang ditetapkan

## Ciri-ciri Sistem
- Antaramuka mesra pengguna (mobile-first)
- Boleh dipasang sebagai aplikasi mudah alih
- Peringatan pengambilan ubat melalui notifikasi
- Boleh digunakan walaupun internet terhad (offline asas)

## Cara Hosting
Sistem ini dihoskan menggunakan GitHub Pages.
